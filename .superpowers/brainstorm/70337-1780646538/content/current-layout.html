关键设计问题：按钮以什么形式呈现？

操作按钮放入 Sidebar 后，有几种不同的呈现方式。请看下面的选项：

A

图标工具栏行

搜索栏上方放一排紧凑的图标按钮（+ ★ ⧉ ↓），hover 显示 tooltip。类似 VS Code 的 Source Control 面板头部的操作图标。

B

Header 内联操作

在 "Profiles (N)" 标题行的右侧放置图标按钮，和现有的排序/批量操作按钮统一排列。

C

主操作按钮 + 下拉菜单

搜索栏上方只放一个醒目的 "新增" 按钮，其余操作（默认、复制、导入）收入 "⋯" 更多菜单。
